iPhenoGram - Getting Started

iPhenoGram is a jQuery widget that utilizes D3.js to create chomosomal ideograms annotated with lines in color at specific base-pair locations or colored base-pair to base-pair regions.  

The code below loads a JSON file and creates a new widget using the data and specifying various options.

$(document).ready(function() {
var data;
d3.json("example_data.json", function(error, json) {
  if (error) return console.warn(error);
  data = json;
  visualizeit();
});

function visualizeit(){
	var chromplot	=	$("#basic").iphenogram({chroms: data.chromsDefault, phenos: data.phenos,
	 	cytoBands: data.cytoDefault, zoom_map: false, chromsOnly: false,
		 chromwidth: 64, circlesize: 28, height: 368, width: 340});
}
});

Displaying only some chromosomes

iPhenoGram has an option to display only some of the chromosomes in the set as below. The chromosomes are drawn to take advantage of the additional space available.

var showChroms = ["7", "10", "11", "12", "18", "21", "X", "Y"];
var fileredPlot	=	$("#filtered").iphenogram({chroms: data.chromsDefault, phenos: data.phenos,
	cytoBands: data.cytoDefault, includedChroms: showChroms,
	chromwidth: 64, circlesize: 28, height: 368, width: 340});

Altering circle size

iPhenoGram has an option to change the size of the phenotypic circles and an option to control the display of the cytogenetic bands (turned off in the example below).

var showChroms = ["7", "10", "11", "12", "18", "21", "X", "Y"];
var fileredPlot	=	$("#filtered").iphenogram({chroms: data.chromsDefault, phenos: data.phenos,
	cytoBands: data.cytoDefault, includedChroms: showChroms, drawCyto: false,
	chromwidth: 64, circlesize: 56, height: 368, width: 340});

Change plot dimensions

iPhenoGram will draw the plot to fill the dimensions given it. Below is the same plot as above but with twice the width given to iPhenoGram.

var showChroms = ["7", "10", "11", "12", "18", "21", "X", "Y"];
var fileredPlot	=	$("#filtered").iphenogram({chroms: data.chromsDefault, phenos: data.phenos,
	cytoBands: data.cytoDefault, includedChroms: showChroms, drawCyto: false,
	chromwidth: 64, circlesize: 56, height: 368, width: 680});
